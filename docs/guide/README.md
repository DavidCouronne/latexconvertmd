# Guide d'utilisation

## Dépendances

+ texsoup
``` bash
pip install texsoup
```
+ dvisvg : à télécharger sur le site [https://dvisvgm.de/](https://dvisvgm.de/)
+ Python 3
+ Une distribution LaTeX


## Installation
```bash
python -m pip install --index-url https://test.pypi.org/simple/ latexconvertmd --upgrade
```
